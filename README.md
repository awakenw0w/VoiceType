# VoiceType

VoiceType - голосовой ИИ, превращающий речь в чёткий, отточенный текст в каждом приложении.

Зажмите выбранную горячую клавишу, скажите текст и отпустите клавишу. VoiceType распознает речь и вставит результат туда, где стоял курсор.

## Возможности

- Настраиваемая горячая клавиша диктовки.
- Быстрое распознавание через нейросеть Groq.
- Локальный режим через `faster-whisper` без отправки аудио в интернет.
- Вставка через буфер обмена с восстановлением предыдущего текстового буфера.
- Защита от небезопасных сочетаний вроде `Alt+Tab`, `Ctrl+Alt+Delete`, `Win+L`.

## Быстрый старт

1. Распакуйте `VoiceType-portable.zip`.
2. Запустите `VoiceType.exe`.
3. В настройках выберите горячую клавишу и тип распознавания.
4. Откройте любое поле ввода, удерживайте горячую клавишу, говорите и отпустите ее.

Для диктовки в приложения, запущенные от администратора, VoiceType тоже может понадобиться запустить от администратора.

## Groq

Для режима `Нейросеть Groq` нужен API-ключ Groq в переменной окружения Windows:

```powershell
setx GROQ_API_KEY "paste-your-key-here"
```

После установки переменной перезапустите VoiceType.

Ключ не хранится в `config.toml`. В настройках приложения показывается только статус, найдена ли переменная `GROQ_API_KEY`.

## Локальный режим

Режим `Локально` использует `faster-whisper`. При первом запуске выбранная модель скачивается на компьютер, после этого распознавание может работать без интернета.

Локальные модели в интерфейсе подписаны по назначению:

- `Самая быстрая (ниже точность)`
- `Быстрая`
- `Сбалансированная`
- `Точная (медленнее)`
- `Максимальная точность`
- `Быстрая большая модель`

## Запуск из исходников

```powershell
cd C:\path\to\VoiceType
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
.\run.ps1
```

## Сборка portable .exe

```powershell
cd C:\path\to\VoiceType
.\build.ps1
```

После сборки появится `VoiceType-portable.zip`. Распакуйте архив и запустите `VoiceType.exe`.
