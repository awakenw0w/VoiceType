# Настройка Groq

Режим `Нейросеть Groq` отправляет завершенную запись диктовки в Groq Speech-to-Text API и вставляет распознанный текст в активное поле ввода.

API-ключ не хранится в `config.toml`. Задайте его как пользовательскую переменную окружения Windows:

```powershell
setx GROQ_API_KEY "paste-your-key-here"
```

После выполнения `setx` перезапустите `VoiceType.exe`.

В настройках VoiceType:

- `Распознавание`: `Нейросеть Groq`
- `Модель нейросети`: `Быстрая нейросеть (рекомендуется)`

Для полностью офлайн-режима выберите `Локально (медленнее, без интернета)`. В этом режиме используется `faster-whisper`, и аудио не отправляется в облако.

Примечания:

- OpenAI-compatible endpoint Groq: `https://api.groq.com/openai/v1/audio/transcriptions`.
- Быстрая модель по умолчанию: `whisper-large-v3-turbo`.
- Если в настройках написано `GROQ_API_KEY: не задан`, VoiceType пока не сможет использовать Groq.
- Слишком тихие записи пропускаются перед распознаванием через `min_audio_rms` в `config.toml`.
